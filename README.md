# Malda Peerless Nursing Home Website

A modern, Apple-inspired, 3D motion multi-specialty hospital website for **Malda Peerless Nursing Home** in Dakshin Jadupur, English Bazar, Malda, West Bengal, India.

## 🚀 Key Features

- **Apple-Inspired Design System**: Frosted glass header, clean typography (`Outfit` & `Inter`), soft shadows, and deep medical blue palette.
- **3D Motion Physics & WebGL Canvas**:
  - Interactive WebGL 3D Medical Helix Canvas (`Apple3DCanvas`).
  - 3D Tilt Cards with specular glare effects (`TiltCard`).
  - Scroll progress bar (`ScrollProgressBar`).
  - Parallax video scroll feel (`useScroll`, `useTransform`).
- **Interactive Booking Engine**: Step-by-step 4-stage appointment wizard generating a printable digital OPD pass with reference ID.
- **Live Hospital Admin Drawer**: Real-time content editor drawer allowing hospital staff to edit phone numbers, email, working hours, ratings, and address with local persistence.
- **Full Hospital Sections**:
  - Hero Section
  - Trust Indicators & Stat Counters
  - Specialized Medical Departments
  - Why Choose Us Pillars
  - About Hospital Mission & Vision
  - Specialist Doctors Directory with Filters
  - Facilities & ICU Showcase
  - Patient Journey Roadmap
  - Infrastructure Gallery & Lightbox
  - Patient Reviews & Testimonials
  - Searchable FAQ Accordion
  - Interactive Contact & Google Maps
  - Mobile Sticky Action Bar (Call, Book, WhatsApp)

## 🛠️ Local Setup & Run

```bash
# Clone the repository
git clone https://github.com/mggamar16-ops/mm-MALDA-PEERLESS-NURSING-HOME.git

# Install dependencies
npm install

# Start local development server
npm run dev

# Build for production
npm run build
```
